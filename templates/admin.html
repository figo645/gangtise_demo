平台发布策略
这套配置只描述平台视角的发布与控开，不再把大V 的日常消息、群发和协同动作混在 Admin 后台里。
租户隔离策略
Admin 负责统一租户隔离策略，保证前台自选股、私信、活动与工作台使用同一套权限边界。
租户研究资料输入源
Admin 在这里配置的是资料输入权限和同步规则，实际的大V 日常使用入口已经拆到独立的 web 工作台。
保存发布
保存后，H5 用户端会按默认主题和功能开关展示；用户自己的主题选择仍然会被本地记住。
